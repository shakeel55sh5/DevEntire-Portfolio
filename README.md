# DevEntire-Portfolio
🛠️ Built With
HTML5: Semantic structure for SEO and accessibility.

CSS3: Custom styling featuring:

Flexbox: Used for responsive grid systems and complex layouts (e.g., the .trainning-items and .FP-Card-Items sections).

CSS Transitions: Smooth hover effects on cards and buttons.

Glassmorphism & Gradients: Modern UI aesthetics using linear-gradient and transparent background blurs.

Sticky Navigation: A user-friendly position: sticky header for easy site navigation.

✨ Key Features
Hero Section: A high-impact "AI-driven future" intro with a gradient text effect and dual call-to-action buttons.

Service Expertise: A 6-card grid showcasing core services like AI Automation, App Development, and Video Editing with interactive hover states.

Project Portfolio: A filtered project showcase (Selected Projects) featuring a grid of high-quality branding and SaaS examples.

Client Testimonials: A dedicated feedback section to build trust and social proof.

Functional Contact Form: A clean, dark-mode form for business inquiries, including custom textarea and input styling.

Responsive Footer: A multi-column footer with newsletter subscription and site-wide navigation links.
